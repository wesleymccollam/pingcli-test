Delete the previous application secret

== Synopsis

Delete the previous secret for a PingOne application before its natural expiry. This is the third step in the rotate → verify → delete-previous workflow.

----
pingcli pingone applications application-secrets delete-previous [flags]
----

== Examples

----
  # Delete the previous application secret early (before its natural expiry)
  pingcli pingone applications application-secrets delete-previous --environment-id <env-id> --application-id <app-id>
----

== Options

----
  -a, --application-id string   The application ID
  -h, --help                    help for delete-previous
  -e, --environment-id string   The PingOne environment ID
----
