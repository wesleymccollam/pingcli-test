Delete a resource

== Synopsis

Delete a resource from a PingOne environment

----
pingcli pingone resources delete [flags]
----

== Examples

----
  # Delete a resource
  pingcli pingone resources delete --environment-id <env-id> --resource-id <resource-id>
----

== Options

----
  -e, --environment-id string   The PingOne environment ID
  -h, --help                    help for delete
  -r, --resource-id string      The resource ID
----
