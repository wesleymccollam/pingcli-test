Delete a user role assignment

== Synopsis

Delete a user role assignment from a PingOne environment

----
pingcli pingone users user-role-assignments delete [flags]
----

== Examples

----
  # Delete a user role assignment
  pingcli pingone users user-role-assignments delete --environment-id <env-id> --user-id <user-id> --role-assignment-id <role-assignment-id>
----

== Options

----
  -e, --environment-id string       The PingOne environment ID
  -h, --help                        help for delete
  -r, --role-assignment-id string   The role assignment ID
  -u, --user-id string              The user ID
----
