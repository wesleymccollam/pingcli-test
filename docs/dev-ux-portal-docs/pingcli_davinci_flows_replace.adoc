Update a DaVinci flow

== Synopsis

Update (replace) a DaVinci flow in a PingOne environment

----
pingcli davinci flows replace [flags]
----

== Examples

----


  # Update a DaVinci flow from a JSON file (--environment-id and --flow-id are still required)
  pingcli davinci flows replace --environment-id <env-id> --flow-id <flow-id> --from-file flow.json

  # Update a DaVinci flow from stdin
  pingcli davinci flows replace --environment-id <env-id> --flow-id <flow-id> --from-file - < flow.json
----

== Options

----
  -e, --environment-id string   The PingOne environment ID
  -h, --help                    help for replace
  -f, --from-file string        Path to a JSON file containing the request body, or "-" to read from stdin.
      --flow-id string          The DaVinci flow ID
----
