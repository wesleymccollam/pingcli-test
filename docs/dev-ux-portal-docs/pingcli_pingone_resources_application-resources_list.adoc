List all application resources

== Synopsis

List all application resources for a resource in a PingOne environment

----
pingcli pingone resources application-resources list [flags]
----

== Examples

----
  # List all application resources for a resource in an environment
  pingcli pingone resources application-resources list --environment-id <env-id> --resource-id <resource-id>
----

== Options

----
  -e, --environment-id string   The PingOne environment ID
  -h, --help                    help for list
  -r, --resource-id string      The parent PingOne Authorize resource ID
      --template string         A Go text/template string. When provided, the command output is rendered through the template instead of the default format. The template receives the command's structured response data. Example: --template '{{.Name}}'
