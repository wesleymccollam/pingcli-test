Generate a sign-on policy assignment JSON template

== Synopsis

Generate a JSON skeleton template for sign-on policy assignment create or replace bodies

----
pingcli pingone applications sop-assignments template [flags]
----

== Examples

----
  # Generate a template and save to a file
  pingcli pingone applications sop-assignments template > assignment.json

  # Edit the template (set priority and signOnPolicy.id), then create the assignment
  pingcli pingone applications sop-assignments template --environment-id <env-id> --application-id <app-id> --from-file assignment.json
Use the JSON template as a starting point:
  1. Run the template command to generate the body skeleton.
  2. Edit the file, replacing placeholder values with real data.
  3. Feed the edited file back to the create or replace action via --from-file.

Example workflow:
  pingcli pingone applications sop-assignments template > body.json
  # edit body.json
  pingcli pingone applications sop-assignments create --from-file body.json
----

== Options

----
  -o, --output-file string   Write the JSON template to PATH instead of stdout. Overwrites any existing file.
  -h, --help                 help for template
----
