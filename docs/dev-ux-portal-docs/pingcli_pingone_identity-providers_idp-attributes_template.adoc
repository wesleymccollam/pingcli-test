Generate an identity provider attribute JSON template

== Synopsis

Generate a JSON skeleton template for identity provider attribute create bodies

----
pingcli pingone identity-providers idp-attributes template [flags]
----

== Examples

----
  # Generate a template and save to a file
  pingcli pingone identity-providers idp-attributes template > body.json
Use the JSON template as a starting point:
  1. Run the template command to generate the body skeleton.
  2. Edit the file, replacing placeholder values with real data.
  3. Feed the edited file back to the create or replace action via --from-file.

Example workflow:
  pingcli pingone identity-providers idp-attributes template > body.json
  # edit body.json
  pingcli pingone identity-providers idp-attributes create --from-file body.json
----

== Options

----
  -o, --output-file string   Write the JSON template to PATH instead of stdout. Overwrites any existing file.
  -h, --help                 help for template
----
