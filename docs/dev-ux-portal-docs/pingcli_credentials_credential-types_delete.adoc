Delete a credential type

== Synopsis

Delete a credential type from a PingOne environment

----
pingcli credentials credential-types delete [flags]
----

== Examples

----
  # Delete a credential type
  pingcli credentials credential-types delete --environment-id <env-id> --credential-type-id <credential-type-id>
----

== Options

----
  -c, --credential-type-id string   The credential type ID
  -h, --help                        help for delete
  -e, --environment-id string       The PingOne environment ID
----
