Create a new application role assignment

== Synopsis

Create a new application role assignment in a PingOne environment

----
pingcli pingone applications application-role-assignments create [flags]
----

== Examples

----


  # Create a new application role assignment from a JSON file
  pingcli pingone applications application-role-assignments create --environment-id <env-id> --application-id <application-id> --from-file role-assignment.json

  # Create a new application role assignment from stdin
  pingcli pingone applications application-role-assignments create --environment-id <env-id> --application-id <application-id> --from-file - < role-assignment.json
----

== Options

----
  -a, --application-id string   The application ID
  -h, --help                    help for create
  -e, --environment-id string   The PingOne environment ID
  -f, --from-file string        Path to a JSON file containing the request body, or "-" to read from stdin.
----
