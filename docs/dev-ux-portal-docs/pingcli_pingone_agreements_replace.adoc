Update an agreement

== Synopsis

Update (replace) an agreement in a PingOne environment

----
pingcli pingone agreements replace [flags]
----

== Examples

----


  # Update an agreement from a JSON file (--environment-id and --agreement-id are still required)
  pingcli pingone agreements replace --environment-id <env-id> --agreement-id <agreement-id> --from-file agreement.json

  # Update an agreement from stdin
  pingcli pingone agreements replace --environment-id <env-id> --agreement-id <agreement-id> --from-file - < agreement.json
----

== Options

----
  -e, --environment-id string   The PingOne environment ID
  -h, --help                    help for replace
  -f, --from-file string        Path to a JSON file containing the request body, or "-" to read from stdin.
  -i, --agreement-id string     The agreement ID
----
