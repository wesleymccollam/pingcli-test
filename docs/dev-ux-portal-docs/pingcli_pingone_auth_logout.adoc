Log out from PingOne

== Synopsis

Log out from PingOne

----
pingcli pingone auth logout [flags]
----

== Examples

----
  # Log out from PingOne (active profile).
    pingcli pingone auth logout

  # Log out from PingOne for a specific profile.
    pingcli pingone auth logout --profile uat

  # Log out from PingOne for all configured profiles.
    pingcli pingone auth logout --all
----

== Options

----
  -a, --all    Log out of all configured profiles (default false)
  -h, --help   help for logout
----
