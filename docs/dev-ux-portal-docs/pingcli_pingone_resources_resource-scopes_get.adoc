Read a specific resource scope

== Synopsis

Read a specific resource scope in a PingOne environment

----
pingcli pingone resources resource-scopes get [flags]
----

== Examples

----
  # Read a specific resource scope
  pingcli pingone resources resource-scopes get --environment-id <env-id> --resource-id <resource-id> --resource-scope-id <scope-id>
----

== Options

----
  -e, --environment-id string      The PingOne environment ID
  -h, --help                       help for get
  -r, --resource-id string         The resource ID
  -s, --resource-scope-id string   The resource scope ID
      --template string            A Go text/template string. When provided, the command output is rendered through the template instead of the default format. The template receives the command's structured response data. Example: --template '{{.Name}}'
