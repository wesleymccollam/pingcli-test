Update an application grant

== Synopsis

Update (replace) an application grant in a PingOne environment

----
pingcli pingone applications application-grants replace [flags]
----

== Examples

----


  # Update an application grant from a JSON file (--environment-id, --application-id, and --application-grant-id are still required)
  pingcli pingone applications application-grants replace --environment-id <env-id> --application-id <app-id> --application-grant-id <grant-id> --from-file application-grant.json

  # Update an application grant from stdin
  pingcli pingone applications application-grants replace --environment-id <env-id> --application-id <app-id> --application-grant-id <grant-id> --from-file - < application-grant.json
----

== Options

----
  -a, --application-id string         The application ID
  -h, --help                          help for replace
  -e, --environment-id string         The PingOne environment ID
  -f, --from-file string              Path to a JSON file containing the request body, or "-" to read from stdin.
  -g, --application-grant-id string   The application grant ID
----
