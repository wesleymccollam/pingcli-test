Update a group

== Synopsis

Update (replace) a group in a PingOne environment

----
pingcli pingone groups replace [flags]
----

== Examples

----
  # Update a group
  pingcli pingone groups replace --environment-id <env-id> --group-id <group-id> --name "Updated Group"
----

== Options

----
  -e, --environment-id string   The PingOne environment ID
  -h, --help                    help for replace
  -g, --group-id string         The group ID
  -n, --name string             The group name (must be unique within an environment)
  -p, --population-id string    The population ID (optional, scopes group to a population)
      --description string      Description of the group
      --display-name string     The display name for the group (defaults to name if not provided)
      --external-id string      A user-defined identifier for the group (for syncing with external systems)
      --source-id string        The source ID for external groups
      --source-type string      The source type for external groups
      --user-filter string      A SCIM filter that determines which users are dynamically added to the group
----
