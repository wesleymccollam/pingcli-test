Create a new population

== Synopsis

Create a new population in a PingOne environment

----
pingcli pingone populations create [flags]
----

== Examples

----
  # Create a new population
  pingcli pingone populations create --environment-id <env-id> --name "My Population" --description "A test population"
----

== Options

----
  -e, --environment-id string              The PingOne environment ID
  -h, --help                               help for create
  -n, --name string                        The population name (must be unique within an environment)
      --alternative-identifiers []string   Alternative identifiers for searching populations
      --default                            Mark as the default population for the environment
      --description string                 Description of the population
      --password-policy-id string          The password policy ID
      --preferred-language string          The language locale for the population (environment default if absent)
      --theme-id string                    The theme ID
----
