Create a new environment

== Synopsis

Create a new environment with the specified configuration

----
pingcli pingone environments create [flags]
----

== Examples

----
  # Create a new SANDBOX environment
  pingcli pingone environments create --name "dev-env" --region EU --type SANDBOX --license-id <license-uuid>

  # Create a new PRODUCTION environment with description
  pingcli pingone environments create --name "prod-env" --region NA --type PRODUCTION --license-id <license-uuid> --description "Production environment"
----

== Options

----
  -d, --description string   A description for the environment
  -h, --help                 help for create
  -l, --license-id string    The license ID for the environment
  -n, --name string          The environment name (must be unique within the organization)
  -r, --region string        The deployment region (AP, AU, CA, EU, NA, SG)
  -t, --type string          The environment type (PRODUCTION or SANDBOX)
      --icon string          The icon identifier for the environment
      --service []string     A product service to enable on the environment (e.g. PING_ONE_DAVINCI). Repeatable.
----
