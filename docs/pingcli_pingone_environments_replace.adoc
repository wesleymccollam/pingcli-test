Update environment

== Synopsis

Update an environment's configuration (full replacement)

----
pingcli pingone environments replace [flags]
----

== Examples

----
  # Update an environment's name
  pingcli pingone environments update --environment-id <environment-uuid> --name "new-name"

  # Update an environment's description
  pingcli pingone environments update --environment-id <environment-uuid> --description "Updated description"
----

== Options

----
  -d, --description string      A description for the environment
  -h, --help                    help for replace
  -e, --environment-id string   The PingOne environment ID
  -n, --name string             The environment name (must be unique within the organization)
  -r, --region string           The deployment region (AP, AU, CA, EU, NA, SG)
  -t, --type string             The environment type (PRODUCTION or SANDBOX)
      --icon string             The icon identifier for the environment
----
