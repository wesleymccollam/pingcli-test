Read a DaVinci variable

== Synopsis

Read a specific DaVinci variable from a PingOne environment

----
pingcli pingone davinci variables get [flags]
----

== Examples

----
  # Read a specific DaVinci variable
  pingcli davinci variables get \\
    --environment-id <env-id> \\
    --variable-id <var-id>
----

== Options

----
  -e, --environment-id string   The PingOne environment ID
  -h, --help                    help for get
  -v, --variable-id string      The variable ID
      --created-at string       The variable creation timestamp
      --template string         A Go text/template string. When provided, the command output is rendered through the template instead of the default format. The template receives the command's structured response data. Example: --template '{{.Name}}'
      --updated-at string       The variable last update timestamp
----
