Update a DaVinci variable

== Synopsis

Update (replace) a DaVinci variable in a PingOne environment

----
pingcli davinci variables replace [flags]
----

== Examples

----
  # Update a DaVinci variable
  pingcli davinci variables replace \\
    --environment-id <env-id> \\
    --variable-id <var-id> \\
    --display-name "Updated Name"
----

== Options

----
  -c, --context string          The variable scope context (company, flow, flowInstance, user)
  -h, --help                    help for replace
  -d, --data-type string        The variable data type (boolean, number, object, secret, string)
  -e, --environment-id string   The PingOne environment ID
  -n, --name string             The variable name (must be unique within the environment)
  -v, --variable-id string      The variable ID
      --display-name string     Display name for the variable (used in UI)
      --flow-id string          ID of the associated DaVinci flow (for flow-scoped variables)
      --max int64               Maximum value for numeric variables (default: 2000)
      --min int64               Minimum value for numeric variables (default: 0)
      --mutable                 Whether the variable value can be modified after creation
      --value string            The initial variable value (JSON encoded)
----
