# `pingcli config profiles use`
Activate and set a custom configuration profile as the in-use profile.

## Synopsis

Activate and set a custom configuration profile as the in-use profile.

```
pingcli config profiles use [flags] [profile-name]
```

## Examples

```
Set an active profile with an interactive prompt to select from an available profile.
    pingcli config profiles use

  Set an active profile with a specific profile name.
    pingcli config profiles use myprofile
```
