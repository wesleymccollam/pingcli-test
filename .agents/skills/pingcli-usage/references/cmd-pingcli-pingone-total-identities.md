# `pingcli pingone total-identities`
Total identity counts

## Synopsis

View total identity counts in a PingOne environment by date

```
pingcli pingone total-identities [flags]
```

## Examples

```
# List all total identity counts by date
  pingcli pingone total-identities --environment-id <env-id> --filter "date ge 2024-01-01"
```
